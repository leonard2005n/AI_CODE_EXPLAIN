# AI_CODE_EXPLAIN Changelog

## [Unreleased]
### Added
- **Multiple AI Actions**: Added "Find Bugs", "Refactor Code", and "Generate Tests" actions alongside "Explain Code".
- **Real-time Streaming**: AI responses now stream into the tool window for immediate feedback.
- **History Management**: Persistent history of previous AI interactions with a searchable dropdown.
- **AI Explainer Tool Window**: Renamed and improved tool window with API key management and navigation.
- **Token Usage Tracking**: Now displays detailed token counts (Input/Output/Total) for every AI request.
- **Gemini Integration**: Updated to use the latest `gemma-4-31b-it` model via Google AI Studio.
- **HTML Rendering**: Enhanced rich text formatting using CommonMark for beautiful AI responses.
- Initial scaffold created from [IntelliJ Platform Plugin Template](https://github.com/JetBrains/intellij-platform-plugin-template)
