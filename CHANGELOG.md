# AI_CODE_EXPLAIN Changelog

## [Unreleased]
### Added
- **AI-Powered Code Explanation**: New action "Explain Code with AI" added to the editor context menu.
- **Gemini Integration**: Seamless integration with Google's Gemini model for high-quality code breakdowns.
- **Dedicated Tool Window**: A new tool window ("MyToolWindow") to display explanations without disrupting the coding flow.
- **Asynchronous Processing**: Background tasks ensure the IDE remains responsive while fetching explanations.
- **HTML Rendering**: Rich text formatting for AI responses, including code snippets and structured lists.
- Initial scaffold created from [IntelliJ Platform Plugin Template](https://github.com/JetBrains/intellij-platform-plugin-template)
